# My casual Hyprland doties yo

> also how do i Lua =/

 ## Requirements
- Arch Linux (or Arch-based distro)
- Hyprland (Lua version)
- Waybar (optional)
- Dolphin
- Kitty
- Dunst
- Rofi



## Setup 

```bash
# CloneClone
git clone https://github.com/LouisRossouw/dotfiles.git
cd YOUR_REPO

# Install
sudo pacman -S hyprland
sudo pacman -S rofi dunst kitty dolphin waybar

# Create symlinks
ln -s ~/YOUR_REPO/hypr ~/.config/hypr
ln -s ~/YOUR_REPO/rofi ~/.config/rofi
ln -s ~/YOUR_REPO/dunst ~/.config/dunst
```

Hopefully it works 🤞
